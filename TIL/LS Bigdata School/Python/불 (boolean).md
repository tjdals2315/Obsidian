# 논리형
- 참 또는 거짓(True/False) 값만 가질 수 있음

## 비교연산자
- 숫자 또는 문자열에 적용
- 문자열에 적용할 경우 글자 순서가 앞에 있는 값이 작은 값(a < b)
	- `==` 같다
	- `!=` 같지 않다 (≠)
	- `<` 작다, 미만
	- `>` 크다, 초과
	- `<=` 작거나 같다, 이하
	- `>=` 크거나 같다, 이상
	```py
	print("가" == "나")
	print("가" != "나")
	print("가" < "나")
	print("가" > "나")
	```

## 논리 연산자
- **not**: 부정의 의미, 결과를 반대로 바꿈 
- **and**: 교집합, 조건을 모두 만족해야 True (`&`)
- **or**: 합집합, 조건 중 하나 만족해도 True(`|`)
	```py
	x=25; print((10<x) & (x<30))
	x=5; print((10<x) | (x<30))
	```
