# 필수 설정
## 1. 편집기
- 새 탭을 위한 기본 화면
	- 편집 화면 or 읽기 화면
- 기본 편집 모드
	- 라이브 미리 보기 or 소스 모드
- Properties in document ^mku0g9
	- 편집 모드일 때 Properties가 어떻게 보일지 설정 
- 행 번호 표시
	- 노트의 좌측에 행마다 번호가 표시됨. 

## 2. 파일 및 링크
- 삭제된 파일
- 새 노트를 만들 위치
- 새 첨부 파일을 만들 위치
- 제외 파일
